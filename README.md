# ``BbAppNewPostSchema``

Data structures for new post creation.

## Overview

The BbAppNewPostSchema library defines the data structures and schemas used in new post creation workflows. It provides type definitions for post creation requests, section selection data, validation rules, and the data models that enable communication between the new post UI components and the underlying content source implementations.
